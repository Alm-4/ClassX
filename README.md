# ClassX
Select assignments/projects from [this](https://www.waikato.ac.nz/study/papers/engee230/2025) class.

Written in Verilog, [this](https://wramp.wand.nz/) assembly language and C. Most of this code was ran on the [Basys3](https://www.amd.com/en/corporate/university-program/aup-boards/digilent-basys3.html) board using Xilinx Vivado or the the respective [toolchain](https://github.com/wandwramp)

## Descriptions
### Chaser
Creates a moving light on the Basys3 FPGAS's leds. The light moves back and forth between any 2 of the leds (done by reading the switches the Basys3 board has for each of the leds) at a varying speed (controlled with buttons on the Basys3 board).

### Assignement1
A FSM design for an elevator.
