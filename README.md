# Engee230
Select assignments/projects from [Engee230](https://www.waikato.ac.nz/study/papers/engee230/2025).

Written in Verilog, the [WRAMP](https://wramp.wand.nz/) assembly language and C. Most of this code was ran on the [Basys3](https://www.amd.com/en/corporate/university-program/aup-boards/digilent-basys3.html) board using Xilinx Vivado or the the [WRAMP toolchain](https://github.com/wandwramp)
